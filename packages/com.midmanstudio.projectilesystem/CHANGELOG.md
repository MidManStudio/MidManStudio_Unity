# Changelog

All notable changes to com.midmanstudio.projectilesystem will be documented here.

## [1.0.0] - Unreleased
### Added
- Rust simulation core: tick, spatial-grid collision, wave/circular movement
- Server-authoritative sim with client prediction and snapshot reconciliation
- 2D and 3D projectile buffers
- Pattern system: Catmull-Rom/Bezier spline editor
- Damage curves (AnimationCurve over normalised distance)
- Impact effects: pooled particles, sprite flipbook, shared emit
- Offline/local-only mode (no NGO dependency)
